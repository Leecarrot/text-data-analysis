## 💡 주제 
'경제' key word를 중심으로 한 text data 수집 및 분석 진행

## 💡 진행과정
1. web scarping을 통한 text data 수집 (naver blog, news)
2. 기본적인 text data 전처리
3. csv file 형태로 변환
4. word-cloud, tf-idf를 통한 분석 진행
